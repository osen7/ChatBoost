# Changelog

All notable changes to this project are documented in this file.

## [Unreleased]

### Added

1. Viewport-priority single-rule render policy.
2. Question-first index navigation with previous/next jump.

1. Latest message protection against aggressive auto-folding.
2. Collapsed/placeholder overlap issues in bottom control area.
3. Detail panel alignment and unexpected jump behavior.

# Changelog

All notable changes to this project are documented in this file.

## [Unreleased]

## [0.2.0] - 2026-03-19

### Added

1. Viewport-priority single-rule render policy.
2. Question-first index navigation with previous/next jump.
3. Runtime pressure and performance metrics in engine stats.
4. Dehydrate memory budget with LRU fallback.
5. Project governance docs: contributing guide and security policy.

### Changed

1. Floating UI simplified to two core actions: `Boost` and `Index`.
2. Optimized list refined into question-focused summary cards.
3. README rewritten to a production-grade open source project format.

### Fixed

1. Latest message protection against aggressive auto-folding.
2. Collapsed/placeholder overlap issues in bottom control area.
3. Detail panel alignment and unexpected jump behavior.

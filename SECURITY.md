# Security Policy

## Supported Scope

ChatBoost is a browser extension that runs locally in the page context.
ChatBoost 是运行在页面上下文中的本地浏览器扩展。

Security-sensitive areas include:
安全敏感区域包括：

1. DOM mutation logic
2. Extension permissions
3. Message rendering and restoration paths
1. DOM 变更逻辑
2. 扩展权限边界
3. 消息渲染与恢复链路

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.
请不要通过公开 Issue 报告安全漏洞。

Send details privately to:
请私下发送到：

- Email: `1275419818@qq.com`

Include:
请包含：

1. Affected version/commit
2. Reproduction steps
3. Impact assessment
4. Suggested mitigation (if available)
1. 受影响版本/提交
2. 复现步骤
3. 影响评估
4. 可能的缓解方案（如有）

## Response Expectations

1. Initial acknowledgment: within 72 hours
2. Triage and severity assessment: as soon as reproducible
3. Fix timeline: based on impact and exploitability
1. 首次响应：72 小时内
2. 分级评估：复现后尽快完成
3. 修复节奏：按影响面与可利用性确定

## Disclosure

After a fix is available, responsible disclosure is encouraged.
修复发布后，欢迎按负责任披露流程公开。
